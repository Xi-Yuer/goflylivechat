技术架构

为性能和可扩展性而构建的现代技术栈

  • Backend: gin, jwt-go, websocket, go.uuid, gorm, cobra
  • Frontend: VueJS, ElementUI
  • Database: MySQL

安装与使用

1. 设置 MySQL 数据库

  • 安装并运行 MySQL（版本 ≥ 5.5）。
  • 创建数据库：
CREATE DATABASE goflychat CHARSET utf8mb4;

配置数据库连接

编辑 config 目录中的 mysql.json：

{
    "Server":"127.0.0.1",
    "Port":"3306",
    "Database":"goflychat",
    "Username":"goflychat",
    "Password":"goflychat"
}

安装和配置 Golang

运行以下命令：

wget https://studygolang.com/dl/golang/go1.20.2.linux-amd64.tar.gz
tar -C /usr/local -xvf go1.20.2.linux-amd64.tar.gz
mv go1.20.2.linux-amd64.tar.gz /tmp
echo "PATH=\$PATH:/usr/local/go/bin" >> /etc/profile
echo "PATH=\$PATH:/usr/local/go/bin" >> ~/.bashrc
source /etc/profile
go version
go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct

下载源代码

在任何目录中克隆仓库：

git clone https://github.com/taoshihan1991/goflylivechat.git
cd goflylivechat

初始化数据库

go run main.go install

运行应用程序

go run main.go server

构建可执行文件

go build -o gochat

运行二进制文件：

Linux: ./gochat server (optional flags: -p 8082 -d)

Windows: gochat.exe server (optional flags: -p 8082 -d)

终止进程

killall gochat

运行后，服务监听 8081 端口。通过 http://[your-ip]:8081 访问。

如需域名访问，配置反向代理到 8081 端口以隐藏端口号。

客服集成

聊天链接

http://127.0.0.1:8081/livechat?user_id=agent
